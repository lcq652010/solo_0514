# 景区门票预订系统

基于 Vue 2 + Element UI 开发的景区门票预订前端系统。

## 功能特性

### 1. 门票类型展示
- 展示景区各种门票类型（成人票、儿童票、老年票、学生票、家庭套票、两日通票）
- 显示门票价格、原价折扣、有效期、限购数量
- 卡片式布局，支持悬停动画效果
- 标签标记热门推荐、优惠等信息

### 2. 在线预订表单
- 选择门票类型、出游日期、购买数量
- 游客信息填写（姓名、手机号、身份证号、备注）
- 表单验证：必填项校验、手机号格式、身份证格式
- 实时计算订单总价
- 订单提交成功后自动跳转订单列表

### 3. 订单列表
- 展示所有订单信息
- 支持按订单状态、订单编号、出游日期筛选
- 订单分页显示
- 订单详情查看
- 订单退款功能
- 订单取消功能

### 4. 检票核销
- 输入取票码验证门票
- 二维码扫描功能占位
- 显示门票详细信息
- 确认核销功能
- 今日核销记录列表

### 5. 客流统计
- 统计卡片展示：今日游客、今日订单、今日营收、剩余门票
- 本周客流趋势图表
- 门票类型销量分布
- 实时客流监控（在园人数、已入园人数、高峰时间、平均游玩时长）

## 技术栈

- **前端框架**: Vue 2.7.14
- **路由管理**: Vue Router 3.6.5
- **UI 组件库**: Element UI 2.15.13
- **构建工具**: Webpack 5
- **HTTP 请求**: Axios

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式运行

```bash
npm run serve
```

项目将在 `http://localhost:8080` 启动

### 生产构建

```bash
npm run build
```

构建产物将输出到 `dist/` 目录

## 页面说明

### 门票类型页面 (/)
- 6种门票类型卡片展示
- 每张卡片包含图片、价格、描述、有效期、限购数量
- 点击"立即预订"跳转到预订表单

### 在线预订页面 (/booking)
- 门票信息展示区
- 预订表单区（门票选择、日期选择、数量选择、游客信息）
- 价格汇总区
- 提交和重置按钮

### 订单列表页面 (/orders)
- 筛选条件区域
- 订单表格
- 分页控件
- 详情弹窗

### 检票核销页面 (/check)
- 取票码输入和验证
- 门票信息展示
- 核销操作
- 今日核销记录

### 客流统计页面 (/statistics)
- 4个统计卡片
- 本周客流趋势表
- 门票销量分布
- 实时监控数据

## 数据存储

项目使用 `localStorage` 存储订单数据，刷新页面后数据不会丢失。

## 浏览器支持

- Chrome (推荐)
- Firefox
- Safari
- Edge
- IE 10+ (需要 polyfill)

## 开发说明

- 所有组件使用单文件组件 (.vue) 格式
- 使用 Element UI 组件库保持样式统一
- 响应式设计，适配不同屏幕尺寸
- 表单验证使用 Element UI 内置验证规则
- 模拟数据位于 `src/mock/data.js`
