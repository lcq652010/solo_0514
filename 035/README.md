# 传统竹编茶则定制订单管理系统 v2.0

## 功能特性

### 1. 必填校验与数值规范
- **必填字段校验**：客户姓名、手机号、款式、尺寸、数量为必填项
- **手机号格式校验**：严格校验中国大陆手机号格式（1开头，11位数字）
- **数量范围校验**：数量必须在 1-1000 之间
- **规格枚举值校验**：竹材种类、篾丝粗细、编织纹样、尺寸规格必须在预设枚举范围内

### 2. 按材料与工艺难度自动计价
系统根据以下因素自动计算价格：
- **竹材种类**：毛竹(20) → 紫竹(65)，不同材质不同价格
- **尺寸规格**：小品(30) → 定制尺寸(100)
- **篾丝粗细**：越细价格越高（0.3mm系数2.5，2.0mm系数0.85）
- **编织纹样复杂度**：人字纹(1.0) → 梅花纹(1.8)
- **价格计算公式**：`(竹材成本 + 尺寸基础价) × 粗细系数 × 纹样系数 × 数量`

### 3. 绑定匠人及工期计算
- **匠人列表**：4位匠人，各有技能等级和日产能
- **工期计算**：根据匠人日产能和订单数量计算总工作天数
- **预计交付日期**：自动计算并存储预计交付时间
- **工序天数**：选竹(1天)、破篾(2天)、蒸煮(1天)、编织(3天)、定型(2天)、打磨(1天)

### 4. 筛选、分页与排序
- **按状态筛选**：待接单/选竹/破篾/蒸煮/编织/定型/打磨/完工
- **按编织纹样筛选**：按不同工艺筛选订单
- **按交付日期筛选**：查询指定日期交付的订单
- **分页功能**：支持自定义页码和每页条数（最大100条）
- **多字段排序**：创建时间、更新时间、价格、交付日期、数量，支持正序/倒序

### 5. 统一接口返回格式
所有接口返回统一格式：
```json
{
  "code": 200,
  "success": true,
  "message": "操作成功",
  "timestamp": "2024-05-16T14:30:00.000",
  "data": {}
}
```

## API 接口列表

### 基础信息
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/` | 系统信息 |
| GET | `/api/statuses` | 获取订单状态列表 |
| GET | `/api/specs` | 获取规格字典（竹材/篾丝/纹样/尺寸） |
| GET | `/api/craftsmen` | 获取匠人列表 |

### 价格计算
| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/price/calculate` | 试算订单价格 |

### 订单管理
| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/orders` | 创建订单（自动计价+计算工期） |
| GET | `/api/orders` | 获取订单列表（支持筛选、分页、排序） |
| GET | `/api/orders/{order_no}` | 获取订单详情 |
| PUT | `/api/orders/{order_no}` | 更新订单（规格变更自动重新计价） |
| PUT | `/api/orders/{order_no}/status` | 更新订单状态 |
| DELETE | `/api/orders/{order_no}` | 删除订单 |

### 统计信息
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/stats` | 获取增强统计（总订单、总营收、按状态/纹样/交付日期统计） |

## 请求参数说明

### 创建订单 POST `/api/orders`
```json
{
  "customer_name": "王五",           // 必填
  "phone": "13900139000",           // 必填，格式校验
  "email": "wangwu@example.com",    // 可选
  "address": "杭州市西湖区",          // 可选
  "style": "传统中式",               // 必填
  "size": "15cm × 8cm",             // 必填
  "quantity": 10,                   // 必填，1-1000
  "bamboo_type": "紫竹",             // 可选，枚举值
  "size_spec": "15cm × 8cm（中号）", // 可选，枚举值
  "strip_thickness": "0.5mm（细）", // 可选，枚举值
  "weaving_pattern": "人字纹",       // 可选，枚举值
  "pattern_description": "边缘装饰", // 可选
  "special_requirements": "精细打磨", // 可选
  "craftsman_id": 1                  // 可选，绑定匠人
}
```

### 获取订单列表 GET `/api/orders`
| 参数 | 类型 | 说明 |
|------|------|------|
| page | int | 页码，默认1 |
| page_size | int | 每页条数，默认10，最大100 |
| status | string | 按状态筛选 |
| weaving_pattern | string | 按编织纹样筛选 |
| delivery_date | string | 按交付日期筛选（YYYY-MM-DD） |
| sort_by | string | 排序字段：created_at/updated_at/total_price/estimated_delivery/quantity |
| sort_order | string | 排序方向：asc/desc，默认desc |
