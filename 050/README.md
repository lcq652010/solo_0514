# 桦树皮笔筒定制订单管理系统后端

基于 Python + Flask + SQLite 实现的传统桦树皮笔筒定制订单管理系统后端（v3.0）。

## 功能特性（v3.0 新增）

### ✅ 必填校验与数值规范
- 客户姓名、手机号、样式、尺寸为必填字段
- 手机号格式校验（11位有效手机号）
- 尺寸数值格式校验（支持带单位如 15cm）
- 枚举值合法性校验（材质、工艺、深度等）

### ✅ 材料与工艺难度自动计价
- 基础价格：200元
- 材料成本（6种树皮材质，80-180元）
- 工艺加成（拼接工艺、雕刻深度）
- 尺寸系数（小号0.8倍，特大号1.5倍）
- 人工成本（4位匠人，150-300元/天）
- 自动计算工期天数

### ✅ 匠人绑定与工期管理
- 4位匠人预设（初级/中级/高级）
- 匠人日薪按级别区分
- 自动计算所需工期（基础1天 + 工艺天数）
- 自动计算人工成本

### ✅ 多条件筛选查询
- 按订单状态筛选（待接单/选皮/蒸煮/裁剪/拼接/雕花/定型/完工）
- 按笔筒风格筛选（圆形/方形/椭圆形/多边形/异形）
- 按树皮材质筛选
- 按交付日期范围筛选

### ✅ 分页与排序
- 支持自定义页码和每页数量
- 支持多字段排序（创建时间/更新时间/价格/交付日期/订单号）
- 支持正序/倒序排列
- 返回完整分页信息

### ✅ 统一接口返回格式
- 所有接口统一返回结构
- 包含 code, success, message, data, timestamp
- 错误信息标准化

## 快速启动

```bash
# 安装依赖
pip install -r requirements.txt

# 运行服务
python app.py
```

服务启动后访问：http://127.0.0.1:5000

## API 接口文档

### 统一返回格式

**成功响应：**
```json
{
  "code": 200,
  "success": true,
  "message": "操作成功",
  "data": { ... },
  "timestamp": "2024-05-16T12:00:00"
}
```

**错误响应：**
```json
{
  "code": 400,
  "success": false,
  "message": "数据验证失败",
  "data": { "errors": [...] },
  "timestamp": "2024-05-16T12:00:00"
}
```

### 订单管理接口

#### 1. 创建订单
```
POST /api/orders
```
**请求参数：**
```json
{
  "customer_name": "张三",
  "customer_phone": "13800138000",
  "customer_address": "北京市朝阳区",
  "penholder_style": "圆形",
  "penholder_size": "大号",
  "carving_pattern": "山水图案",
  "special_requirements": "希望能刻上名字",
  "bark_material": "白桦皮",
  "size_height": "15cm",
  "size_diameter": "8cm",
  "size_thickness": "2mm",
  "splicing_tech": "平接",
  "carving_position": "正面",
  "carving_depth": "中雕(2-4mm)",
  "carving_detail": "线条流畅，层次分明",
  "craftsman_id": 1,
  "delivery_date": "2024-05-25"
}
```

#### 2. 获取订单列表（支持筛选分页）
```
GET /api/orders?status=待接单&style=圆形&material=白桦皮&delivery_from=2024-05-01&delivery_to=2024-05-31&page=1&page_size=10&sort_by=price&sort_order=desc
```

**返回数据：**
```json
{
  "list": [...],
  "pagination": {
    "page": 1,
    "page_size": 10,
    "total": 100,
    "total_pages": 10
  }
}
```

#### 3. 获取单个订单详情
```
GET /api/orders/{order_id}
```

#### 4. 编辑订单
```
PUT /api/orders/{order_id}
```
支持部分字段更新，价格相关字段变更时会自动重新计价。

#### 5. 更新订单状态
```
PUT /api/orders/{order_id}/status
{"status": "选皮"}
```

#### 6. 删除订单
```
DELETE /api/orders/{order_id}
```

#### 7. 试算订单价格
```
POST /api/orders/calculate
```
传入订单参数，返回预估价格和工期，不实际创建订单。

### 数据字典接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/statuses` | GET | 获取订单状态列表 |
| `/api/materials` | GET | 获取树皮材质列表（含价格） |
| `/api/splicing-techniques` | GET | 获取拼接工艺列表（含价格、工期） |
| `/api/carving-depths` | GET | 获取雕刻深度列表（含价格、工期） |
| `/api/styles` | GET | 获取笔筒样式列表 |
| `/api/sizes` | GET | 获取笔筒尺寸列表（含价格系数） |
| `/api/craftspeople` | GET | 获取匠人列表（含级别、日薪） |
| `/api/process-specs` | GET | 获取完整工艺规格（全部字典） |

```
总价 = (基础价 + 材料费 + 工艺费) × 尺寸系数 + 人工费
```

## 数据库结构（v3.0）

orders 表新增字段：
- `price` - 订单总价
- `material_cost` - 材料成本
- `process_cost` - 工艺成本
- `labor_cost` - 人工成本
- `craftsman_id` - 匠人ID
- `craftsman_name` - 匠人姓名
- `work_days` - 所需工期
- `delivery_date` - 交付日期
