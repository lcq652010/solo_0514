## 1. 产品概述

缂丝扇面定制系统是一个传统工艺数字化定制平台，为客户提供个性化缂丝扇面定制服务，同时为管理员提供订单生产全流程管理功能。

- 主要目的：实现缂丝扇面定制流程的线上化，提升客户定制体验和生产管理效率
- 目标用户：有个性化扇面定制需求的客户、缂丝工坊管理员

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 客户 | 直接访问下单 | 填写定制表单、提交订单、查看订单状态 |
| 管理员 | 登录后台 | 查看订单列表、管理生产工序、更新订单状态 |

### 2.2 功能模块

1. **客户下单页**：丝线材质选择、扇面尺寸输入、纹样题材选择、织造密度选择、包边方式选择、订单提交
2. **管理员订单页**：订单列表展示、生产工序追踪、工序进度更新、订单状态管理

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 客户下单页 | 材质选择 | 提供多种丝线材质选项，支持单选 |
| 客户下单页 | 尺寸输入 | 支持扇面宽度、高度输入，带单位提示 |
| 客户下单页 | 纹样题材 | 提供多种传统纹样主题选择 |
| 客户下单页 | 织造密度 | 提供不同织造密度规格选项 |
| 客户下单页 | 包边方式 | 提供多种包边工艺选择 |
| 客户下单页 | 订单提交 | 表单验证、提交成功反馈 |
| 管理员订单页 | 订单列表 | 展示所有订单基本信息，支持筛选 |
| 管理员订单页 | 工序管理 | 8道生产工序进度展示与更新 |

## 3. 核心流程

客户选择定制参数 → 填写订单信息 → 提交订单 → 系统生成订单 → 管理员查看订单 → 更新各工序进度 → 订单完工

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：中国传统色彩，以赭石色、米色为主，体现传统工艺气质
- 辅色调：墨绿色、藏青色作为点缀
- 按钮风格：圆角矩形，传统纹样装饰边框
- 字体：宋体/思源宋体为主，体现文化底蕴
- 布局风格：卡片式布局，留白充足，雅致简约
- 整体风格：新中式，传统与现代融合

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|---------|
| 客户下单页 | 表单区域 | 分组卡片、下拉选择器、数字输入框、单选按钮组 |
| 客户下单页 | 提交区域 | 总价预览、提交按钮、成功提示弹窗 |
| 管理员订单页 | 订单列表 | 表格、状态标签、筛选器、分页 |
| 管理员订单页 | 工序进度 | 时间线组件、步骤条、状态更新按钮 |

### 4.3 响应式

- 桌面端优先设计，1200px 以上最佳展示
- 平板端自适应，表单两列变单列
- 移动端单列布局，优化触控区域

### 4.4 装饰元素

- 传统云纹、回纹作为装饰边框
- 水墨质感背景纹理
- 扇面形状图标和装饰元素
