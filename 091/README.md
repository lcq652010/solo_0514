# 体检中心管理系统

基于 Vue 2 + Element UI 开发的体检中心前端管理系统。

## 功能特性

- 体检套餐列表：展示各类体检套餐，支持按名称、类型筛选
- 在线预约：填写预约信息，选择套餐和时间
- 个人体检记录：查看历史体检记录，支持状态筛选
- 报告上传：上传体检报告文件，支持多种格式
- 报告查询：查询和管理已上传的体检报告

## 技术栈

- **框架**: Vue 2.7.x
- **UI 组件库**: Element UI 2.x
- **路由**: Vue Router 3.x
- **构建工具**: Webpack 5.x
- **HTTP 客户端**: Axios

## 项目结构

```
├── src/
│   ├── views/              # 页面组件
│   │   ├── PackageList.vue    # 体检套餐列表

│   │   ├── ExamRecords.vue    # 体检记录
│   │   ├── ReportUpload.vue   # 报告上传
│   │   └── ReportQuery.vue    # 报告查询
│   ├── router/             # 路由配置
│   │   └── index.js
│   ├── App.vue             # 根组件
│   └── main.js             # 入口文件
├── index.html              # HTML 模板

```

## 安装依赖

```bash
npm install
```

## 开发运行

```bash
npm run dev
```

项目将在 `http://localhost:8080` 启动。

## 生产构建

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

## 页面说明

### 1. 体检套餐列表
- 卡片式展示各类体检套餐
- 支持按套餐名称、套餐类型搜索
- 显示套餐原价和优惠价
- 点击"立即预约"跳转到预约页面

### 2. 在线预约
- 选择体检套餐
- 选择预约日期和时间段
- 填写个人信息（姓名、性别、手机号、身份证号等）
- 表单必填项校验
- 支持备注信息填写

### 3. 个人体检记录
- 表格展示体检记录列表
- 支持按日期范围、状态筛选
- 查看记录详情
- 支持预约取消、报告下载操作

### 4. 报告上传
- 关联体检记录
- 选择报告类型（主报告、检验报告、影像报告等）
- 拖拽上传文件（支持 PDF、Word、图片格式）
- 已上传报告列表展示和管理

### 5. 报告查询
- 多条件搜索（记录编号、姓名、手机号、报告类型、日期）
- 表格展示报告列表
- 支持批量选择和批量删除
- 查看报告详情和下载报告

## 浏览器支持

- Edge

## 开发说明

- 所有页面均使用 Element UI 组件库开发
- 使用模拟数据展示功能，实际项目中需要对接后端 API
- 表单校验使用 Element UI 内置的校验规则
- 分页功能已实现，可根据实际需求调整
