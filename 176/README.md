# 锡制茶叶罐定制系统

基于 Vue2 + ElementUI 实现的锡制茶叶罐定制平台，包含客户下单与管理员订单管理模块。

## 功能特性

### 客户下单页面
- **基本规格配置**
  - 锡料纯度选择：97% 纯锡、99% 高纯锡、99.9% 精锡
  - 罐身高度输入（50-200mm）
  - 口径尺寸输入（30-150mm）

- **工艺配置**
  - 纹饰风格选择：光面无纹、传统祥云、山水意境、梅兰竹菊
  - 密封配置选择：标准硅胶圈、双层密封圈、真空密封盖

- **客户信息**
  - 姓名、联系电话、备注说明
  - 表单验证功能

### 管理员订单管理
- **订单列表展示**：可折叠的订单详情卡片
- **生产工序流程管理**（8道工序）：
  1. 熔锡
  2. 锻打
  3. 车旋
  4. 錾刻
  5. 抛光
  6. 密封装配
  7. 质检
  8. 完工

- 工序推进与重置功能
- 订单状态实时显示

## 安装与运行

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run serve
```

项目将在 `http://localhost:8080` 启动

### 构建生产版本
```bash
npm run build
```

## 技术栈

- **Vue 2.6.x** - 渐进式 JavaScript 框架
- **Vue Router 3.x** - 官方路由管理器
- **Element UI 2.x** - 桌面端组件库
- **Vue CLI 4.x** - 脚手架工具

## 页面说明

### 客户下单页面 (`/`)
用户在此页面填写定制需求，包括规格、工艺、个人信息等，提交后生成订单。

### 订单管理页面 (`/admin`)
管理员在此查看所有订单，管理生产工序进度，可逐步推进工序或重置工序。

## 使用说明

1. 在客户下单页面填写所有必填项（带 * 号）
2. 点击"提交定制订单"生成订单
3. 切换到"订单管理"页面查看订单详情
4. 点击"完成当前工序"推进生产进度
5. 8道工序全部完成后订单状态变为"已完成"

## 注意事项

- 订单数据存储在内存中，刷新页面后数据会丢失
- 如需持久化存储，可集成后端 API 或本地存储
- 手机号格式验证为中国大陆手机号（1开头的11位数字）
