# 二手书回收售卖系统

基于 Vue 2 + Element UI 开发的二手书回收售卖管理系统。

## 功能特性

### 1. 书籍录入表单
- 完整的书籍信息录入表单（书名、作者、ISBN、分类、成色、原价、售价、描述等）
- 卖家信息录入（姓名、电话）
- 表单必填项校验
- 自动计算回收价格

### 2. 回收定价页
- 回收申请列表展示
- 支持按书名/作者搜索
- 支持按状态筛选（待审核、已完成、已拒绝）
- 可编辑回收价格
- 审核通过/拒绝操作
- 回收详情查看

### 3. 书籍上架列表
- 书籍列表展示
- 支持多条件搜索（书名、作者、ISBN）
- 支持分类和状态筛选
- 上架/下架操作
- 书籍详情查看
- 删除书籍

### 4. 订单购买页
- 左侧：可购买书籍列表
- 右侧：购物车和订单表单
- 加入购物车、调整数量、移除商品
- 订单信息填写和提交
- 订单列表展示
- 发货、完成订单操作

### 5. 回收记录列表
- 回收记录列表展示
- 支持多条件搜索（书名、作者、卖家）
- 支持状态和日期筛选
- 回收详情查看

## 技术栈

- **Vue 2.7.x**: 前端框架
- **Vue Router 3.x**: 路由管理
- **Element UI 2.x**: UI 组件库
- **Vite**: 构建工具
- **LocalStorage**: 数据持久化

## 安装和运行

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

项目将在 `http://localhost:3000` 启动。

### 3. 构建生产版本

```bash
npm run build
```

### 4. 预览生产构建

```bash
npm run preview
```

## 页面说明

| 页面路径 | 页面名称 | 功能 |
|---------|---------|------|
| /book-entry | 书籍录入 | 录入回收书籍信息 |
| /pricing | 回收定价 | 审核回收申请，设置价格 |
| /book-list | 书籍上架 | 管理书籍上架/下架 |
| /order | 订单购买 | 购买书籍、管理订单 |
| /recycle-records | 回收记录 | 查看所有回收记录 |

## 数据说明

系统使用 LocalStorage 进行数据持久化，首次访问时会自动初始化模拟数据。

包含的模拟数据：
- 5 本不同状态的书籍
- 3 个不同状态的订单
- 5 条不同状态的回收记录
