# 影院自助终端运维管理系统

基于 Vue2 + Element UI 实现的影院自助取票购票终端运维管理系统前端页面。

## 功能特性

- **设备列表展示**：完整展示设备编号、影院门店、摆放区域、设备运行状态
- **分页功能**：按设定数量加载对应范围设备数据，支持切换每页显示数量（10/20/50/100条）
- **设备状态颜色标记**：
  - 运行中（绿色标签）
  - 离线（灰色标签 + 灰色行背景）
  - 故障（红色标签 + 红色行背景）
  - 维护中（橙色标签 + 橙色行背景）
- **异常状态醒目行样式**：故障、离线、维护中的设备自动应用差异化行背景色，状态变动时样式自动同步
- **运行状态筛选**：支持按设备运行状态（运行中/离线/故障/维护中）筛选设备列表
- **综合关键字搜索**：支持输入关键字匹配设备编号、影院名称、摆放区域、设备状态
- **模块运行状态展示**：列表展示身份读取、票据打印、支付交互三个模块的运行状态（正常/警告/异常）
- **故障上报弹窗**：
  - 设备选择控件（支持搜索过滤）
  - 故障类型选择（硬件/软件/网络/打印/其他）
  - 故障描述（最多500字符，带字数统计）
  - 上报人、联系电话字段
- **完善的提交验证**：上报前判断设备选择情况，未选择则弹出提示并阻断提交；所有必填项验证，手机号格式校验，故障描述长度校验
- **工单自动编号**：自动生成格式为 `WO + 日期 + 4位随机数` 的工单编号
- **状态自动同步**：故障上报成功后，对应设备状态自动变更为"故障"，列表样式即时更新

## 安装和运行

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

服务器启动后，访问 `http://localhost:8080` 即可查看页面。

### 3. 构建生产版本

```bash
npm run build
```

## 使用说明

### 搜索设备
1. 在搜索栏输入关键字（支持设备编号、影院名称、摆放区域、设备状态）
2. 点击「搜索」按钮或按回车键进行筛选
3. 点击「重置」按钮清空搜索条件

### 翻页浏览
1. 点击分页器的页码切换不同批次的设备列表
2. 可通过每页条数切换器调整每页显示设备数量
3. 翻页后自动加载对应批次的设备信息

### 故障上报
1. 点击页面右上角的「故障上报」按钮，或点击设备列表中的「上报故障」按钮
2. 在弹窗中选择需要上报故障的设备（支持搜索过滤）
3. 选择故障类型（硬件/软件/网络/打印/其他）
4. 填写故障描述（至少10个字符，最多500字符）
5. 填写上报人姓名和联系电话
6. 点击「确定」提交，系统会自动生成工单编号并提示成功

## 数据说明
