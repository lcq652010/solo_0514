# 会员管理系统

基于 Vue2 + ElementUI 开发的会员管理前端页面系统。

## 功能特性

### 1. 会员信息列表页
- 会员信息表格展示（会员编号、姓名、手机号、会员等级、账户余额、积分、状态、注册时间等）
- 多条件搜索功能（会员编号、姓名、手机号、会员状态）
- 分页功能
- 操作按钮（查看详情、充值、消费）

### 2. 会员详情页
- 会员基本信息完整展示
- 账户余额、累计充值、累计消费、会员积分等数据展示
- 快捷操作按钮（充值、消费、编辑信息）
- 最近交易记录展示

### 3. 充值表单页
- 会员选择（支持搜索过滤）
- 充值金额输入（支持快捷金额）
- 多种支付方式选择
- 表单必填项验证
- 充值后余额预览

### 4. 消费扣款表单页
- 会员选择（支持搜索过滤）
- 消费金额输入（支持快捷金额）
- 多种支付方式选择
- 表单必填项验证
- 消费后余额预览及余额不足提示

### 5. 交易流水记录页
- 交易流水表格展示
- 多条件筛选（交易单号、会员信息、交易类型、支付方式、交易时间范围）
- 分页功能
- 交易金额正负区分显示

## 技术栈

- Vue 2.6.x
- Vue Router 3.5.x
- Element UI 2.15.x
- Moment.js
- Sass

## 安装与运行

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run serve
```

项目将在 `http://localhost:8080` 启动。

### 3. 构建生产版本

```bash
npm run build
```

## 使用说明

### 页面导航

- 顶部导航栏可切换 "会员列表" 和 "交易流水" 页面
- 在会员列表页可点击操作按钮进入详情页、充值页或消费页
- 各页面均有返回按钮可返回上一级页面

### 表单验证

- 充值/消费表单均有必填项验证
- 金额必须大于 0
- 提交前会弹出确认对话框

### 模拟数据说明

系统预置了 5 个会员和 6 条交易记录的模拟数据，便于测试功能。

## 功能截图说明

### 会员列表
- 展示所有会员的基本信息
- 支持按会员编号、姓名、手机号、状态筛选
- 会员余额突出显示
- 不同会员等级用不同颜色标签区分

### 会员详情
- 完整展示会员的所有信息
- 显示最近 10 条交易记录

- 表单验证确保数据完整性
- 实时计算并预览操作后的余额

### 交易流水
- 所有交易记录一览
- 支持多维度筛选查询
- 充值用绿色显示，消费用红色显示
