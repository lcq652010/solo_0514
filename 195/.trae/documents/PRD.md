## 1. 产品概述
社区服务自助办事终端运维管理系统，用于管理社区内的自助服务终端设备，提供设备监控、故障上报和工单管理功能。
- 主要面向社区运维管理人员，解决设备分散、故障难以及时发现和处理的问题
- 提升社区服务设备的运维效率，保障设备正常运行

## 2. 核心功能

### 2.1 功能模块
1. **设备列表页面**：设备信息展示、分页、搜索、状态展示
2. **故障上报弹窗**：设备选择、故障说明填写、工单提交
3. **工单管理**：自动生成工单编号

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 设备列表页面 | 搜索区域 | 支持按设备编号、社区名称模糊搜索 |
| 设备列表页面 | 设备列表 | 展示设备编号、所属社区、安放点位、设备运行状态 |
| 设备列表页面 | 分页区域 | 完整的分页功能，支持页码跳转和每页条数切换 |
| 设备列表页面 | 故障上报按钮 | 点击打开故障上报弹窗 |
| 故障上报弹窗 | 设备选择 | 下拉选择要上报故障的设备 |
| 故障上报弹窗 | 故障说明 | 文本框填写故障详情 |
| 故障上报弹窗 | 提交工单 | 自动生成工单编号并提交 |

## 3. 核心流程
用户进入设备列表页面，查看设备状态；如需上报故障，点击故障上报按钮，选择设备并填写故障说明，提交后系统自动生成工单编号，完成故障上报。

    D --> E["填写故障说明"]

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：蓝色系（#1890ff），代表专业和信赖
- **状态颜色**：
  - 运行中：绿色（#52c41a）
  - 故障：红色（#ff4d4f）
  - 离线：灰色（#8c8c8c）
  - 维护中：橙色（#fa8c16）
- **按钮风格**：圆角按钮，hover时有过渡效果
- **字体**：使用系统默认字体，清晰易读
- **布局风格**：卡片式布局，顶部导航+主内容区
- **图标**：使用简洁的线性图标

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 设备列表页面 | 搜索区域 | 输入框、搜索按钮、重置按钮 |
| 设备列表页面 | 操作栏 | 故障上报按钮、数据统计 |
| 设备列表页面 | 列表表格 | 表头、数据行、状态标签 |
| 设备列表页面 | 分页区域 | 页码、上一页/下一页、每页条数选择 |
| 故障上报弹窗 | 弹窗主体 | 标题栏、表单、底部操作按钮 |

### 4.3 响应式
桌面端优先，适配常见分辨率（1366×768及以上），主要操作区域居中显示。
