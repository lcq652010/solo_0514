# 培训机构排课考勤系统

基于 Vue 2 + Element UI 开发的培训机构排课考勤管理系统。

## 功能特性

### 1. 课程列表页
- 📋 课程信息展示（编号、名称、类型、课时、学费、老师、状态）
- 🔍 支持按课程名称和状态搜索
- ➕ 新增课程（表单校验）
- ✏️ 编辑课程
- 🗑️ 删除课程（二次确认）
- 🚦 课程启用/停用切换
- 📄 分页展示

### 2. 班级排课表单
- 📅 班级排课信息管理
- 🏫 按课程、月份筛选
- ⏰ 上课时间选择（日期+时段）
- 👨‍🏫 授课老师分配
- 🏢 教室选择
- 📊 班级人数上限设置
- 👁️ 排课详情查看

### 3. 学员花名册
- 👥 学员信息管理（学号、姓名、性别、年龄、手机、邮箱、班级）
- 🔍 支持按姓名、手机号、班级搜索
- 📥 Excel导出功能
- 📈 学员出勤率可视化展示
- 📝 学员状态管理（在读、休学、毕业）
- 详情弹窗展示学员完整信息

### 4. 上课考勤打卡页
- ✅ 实时时间显示
- 🏫 按班级和日期筛选
- 单个学员打卡（出勤/迟到/缺勤/请假）
- 批量打卡功能
- 📊 考勤状态实时统计（出勤/迟到/缺勤/请假人数）
- 💬 备注信息填写

### 5. 考勤统计页
- 📊 汇总数据卡片展示（学员总数、出勤人次、平均出勤率、已上课次）
- 📈 月度考勤趋势表格
- 🥧 考勤状态分布可视化
- 🏆 学员考勤排行榜（支持按出勤率/出勤次数排序）
- 🎖️ 排名徽章（金/银/铜）

## 技术栈

- **前端框架**: Vue 2.6.14
- **UI组件库**: Element UI 2.15.13
- **路由**: Vue Router 3.5.3
- **样式**: SCSS
- **HTTP请求**: Axios 0.27.2

## 安装与运行

### 环境要求
- Node.js >= 12.0.0
- npm >= 6.0.0

### 安装依赖
```bash
npm install
```

### 开发模式运行
```bash
npm run serve
```
访问 http://localhost:8080

### 生产环境构建
```bash
npm run build
```

## 表单校验规则

- **课程编号**: 必填，只能包含字母和数字
- **课程名称**: 必填，长度2-50字符
- **授课老师**: 必填
- **手机号**: 必填，11位有效手机号
- **班级名称**: 必填，长度2-30字符
- **上课日期/时间**: 必填
