# 物资领用管理系统

基于 Vue 2 + Element UI 开发的物资领用管理前端系统。

## 功能特性

- **物资列表展示**：查看所有可领用物资，支持按名称和类别搜索，显示库存状态
- **领用申请表单**：提交物资领用申请，支持选择物资、填写数量、用途等信息
- **归还登记表单**：登记物资归还，支持选择领用记录、填写物资状态
- **领用记录查询**：查看所有领用记录，支持搜索、筛选，显示申请状态统计

## 技术栈

- Vue 2.6.x
- Vue Router 3.x
- Element UI 2.x
- SCSS

## 安装与运行

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run serve
```

默认启动在 http://localhost:8080

### 3. 构建生产版本

```bash
npm run build
```

## 页面说明

### 1. 物资列表 (/materials)
- 显示所有物资的详细信息
- 支持按物资名称、类别搜索
- 显示库存状态（正常、库存低、无库存）
- 可直接点击"领用"跳转到申请页面

### 2. 领用申请 (/apply)
- 选择物资、填写领用数量
- 输入申请人、部门、用途等信息
- 选择申请日期和预计归还日期
- 表单验证确保必填项完整

### 3. 归还登记 (/return)
- 选择已批准的领用记录
- 自动显示领用信息
- 填写实际归还日期和数量
- 选择物资状态（完好无损、轻微磨损、损坏、丢失）

### 4. 领用记录 (/records)
- 统计卡片显示各状态数量
- 支持按物资名称、状态、日期范围搜索
- 表格展示完整记录信息
- 支持查看详情、撤销申请、快速归还

## 注意事项

1. 当前使用模拟数据，实际使用时需对接后端 API
2. 物资状态会影响可用性，无库存物资无法领用
3. 只有待审批状态的申请可以撤销
4. 只有已批准状态的记录可以进行归还操作
5. 物资损坏或丢失时会有相应提示
