## 1. 产品概述

宣纸笺定制管理系统，为传统宣纸制作行业提供数字化定制服务平台，包含客户在线下单与管理员生产流程管理两大核心模块。

- 面向客户：提供宣纸品类、尺寸、帘纹、描金、包装等个性化定制选项，实现一键下单
- 面向管理员：实现从选纸到完工的全流程工序跟踪管理，提升生产效率

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 客户 | 直接访问 | 浏览定制选项、填写订单信息、提交订单 |
| 管理员 | 密码登录 | 查看订单列表、管理生产工序、更新订单状态 |

### 2.2 功能模块

1. **客户下单页**：宣纸品类选择、尺寸规格输入、帘纹款式选择、描金工艺选择、包装方式选择、订单提交
2. **管理员登录页**：管理员身份验证
3. **订单管理页**：订单列表展示、生产工序进度管理、工序状态更新

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|----------|----------|
| 客户下单页 | 品类选择 | 下拉选择宣纸品类（生宣、熟宣、半生熟等） |
| 客户下单页 | 尺寸规格 | 输入长/宽尺寸，支持常用规格快捷选择 |
| 客户下单页 | 帘纹款式 | 单选帘纹图案（单丝路、双丝路、罗纹、龟纹等） |
| 客户下单页 | 描金工艺 | 多选描金样式（描金龙、描金凤、描金云纹等） |
| 客户下单页 | 包装方式 | 单选包装类型（简装、锦盒装、卷轴装） |
| 客户下单页 | 提交订单 | 表单验证、提交订单、显示订单编号 |
| 订单管理页 | 订单列表 | 展示所有订单，支持按状态筛选 |
| 订单管理页 | 工序流程 | 可视化展示8道生产工序（选纸→裁切→水印→描金→压平→包装→质检→完工） |
| 订单管理页 | 状态更新 | 点击工序按钮更新生产进度 |

## 3. 核心流程

### 3.1 客户下单流程
客户选择宣纸品类 → 输入尺寸规格 → 选择帘纹款式 → 选择描金工艺 → 选择包装方式 → 提交订单 → 系统生成订单号

```mermaid
flowchart LR
    A["品类选择"] --> B["尺寸输入"]
    B --> C["帘纹选择"]
    C --> D["描金选择"]
    D --> E["包装选择"]
    E --> F["提交订单"]
    F --> G["生成订单号"]
```

### 3.2 管理员生产流程
管理员登录 → 查看待处理订单 → 依次完成8道工序 → 订单完工

```mermaid
flowchart LR
    A["选纸"] --> B["裁切"]
    B --> C["水印"]
    C --> D["描金"]
    D --> E["压平"]
    E --> F["包装"]
    F --> G["质检"]
    G --> H["完工"]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：米白色 (#F5F0E6) 为背景，仿宣纸质感；朱红色 (#C41E3A) 为强调色，体现传统工艺特色
- **辅助色**：深棕色 (#5D4E37) 文字，墨黑色 (#1A1A1A) 标题
- **按钮风格**：圆角矩形，淡雅边框，悬停有渐变效果
- **字体**：标题使用"楷体"或"宋体"，正文使用"微软雅黑"，体现传统文化气质
- **布局风格**：卡片式分组布局，表单区域清晰分隔，留白适中
- **装饰元素**：融入水墨纹理、祥云图案等传统元素作为背景点缀

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|----------|---------|
| 客户下单页 | 头部区域 | 品牌标题、副标题、传统纹样装饰 |
| 客户下单页 | 表单区域 | 分组卡片、标签+控件布局、单选/多选框组 |
| 客户下单页 | 底部区域 | 总价预览、提交按钮 |
| 订单管理页 | 顶部导航 | 系统名称、管理员信息、退出按钮 |
| 订单管理页 | 订单列表 | 表格展示、状态标签、操作按钮 |
| 订单管理页 | 工序面板 | 步骤条、工序按钮、完成状态标识 |

### 4.3 响应式设计
- 桌面端优先设计，主内容区域宽度 1200px 居中
- 表单分组采用两列布局，在小屏幕下自动切换为单列
- 生产工序步骤条在移动端可横向滚动

## 5. 交互设计

- 表单验证：必填项实时校验，错误提示清晰友好
- 工序更新：点击工序按钮后有动画反馈，状态颜色变化
- 订单提交：加载状态提示，成功后显示订单编号和成功动画
- 筛选功能：订单列表支持按状态快速筛选
