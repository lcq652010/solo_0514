# 市民服务中心自助证明打印终端运维管理系统

基于 Vue2 + Element UI 实现的前端运维管理系统。

## 功能特性

### 1. 设备列表展示
- 设备编号
- 服务网点
- 摆放区域
- 设备状态（正常/故障/离线/维护中）
- 最后在线时间

### 2. 分页功能
- 独立分页模块
- 支持每页显示条数设置（10/20/50/100）
- 支持页码跳转

### 3. 搜索功能
- 按设备编号模糊检索
- 按网点名称模糊检索

### 4. 设备状态颜色标记
- 正常：绿色
- 故障：红色
- 离线：灰色
- 维护中：橙色

### 5. 故障上报功能
- 弹窗形式
- 自动生成工单编号（格式：WO + 年月日 + 4位随机数）
- 可选择设备（支持筛选）
- 录入故障说明
- 表单验证

## 运行方式

### 方式一：直接打开（推荐）
直接在浏览器中打开 `index.html` 文件即可运行。

### 方式二：使用 HTTP 服务器（推荐）
```bash
# 使用 Node.js
npm start
# 或
npx http-server -p 8000 -o

# 使用 Python
python -m http.server 8000
# 然后在浏览器访问 http://localhost:8000
```

## 技术栈

- **Vue 2.6.14** - 前端框架
- **Element UI 2.15.13** - UI 组件库
- **CDN 引入** - 无需构建工具，开箱即用

## 使用说明

1. **搜索设备**：在搜索栏输入设备编号或网点名称，点击搜索按钮
2. **故障上报**：点击"故障上报"按钮，选择设备并填写故障说明后提交
3. **分页浏览**：使用底部分页组件浏览设备列表
4. **状态查看**：设备状态使用不同颜色标记，一目了然

## 注意事项

- 项目使用 CDN 引入依赖，需要联网运行
- 数据为模拟数据，实际使用时请对接后端 API
