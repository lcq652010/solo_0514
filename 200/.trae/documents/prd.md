## 1. 产品概述

石砚定制前端系统是一个为传统文房四宝行业打造的数字化定制平台，支持客户在线下单定制石砚，并为管理员提供订单全流程生产管理能力。

* 核心价值：将传统石砚制作工艺与互联网结合，实现定制流程可视化、生产工序数字化

* 目标用户：文房四宝爱好者、书画家、收藏家，以及砚台生产企业管理人员

## 2. 核心功能

### 2.1 用户角色

| 角色  | 登录方式     | 核心权限          |
| --- | -------- | ------------- |
| 客户  | 直接访问     | 填写定制信息、提交订单   |
| 管理员 | 直接访问管理页面 | 查看订单列表、跟踪生产工序 |

### 2.2 功能模块

1. **客户下单页**：砚石材选择、尺寸输入、雕刻样式选择、砚池形制选择、订单提交
2. **管理员订单页**：订单列表展示、生产工序进度跟踪、工序状态更新

### 2.3 页面详情

| 页面名称   | 模块名称   | 功能描述                                  |
| ------ | ------ | ------------------------------------- |
| 客户下单页  | 砚石材选择  | 下拉选择砚石材质（端砚石、歙砚石、洮砚石、澄泥砚石等）           |
| 客户下单页  | 尺寸输入   | 数字输入框输入砚台长度和宽度（单位：厘米）                 |
| 客户下单页  | 雕刻样式选择 | 单选或下拉选择雕刻风格（素面、云龙纹、山水纹、梅兰竹菊等）         |
| 客户下单页  | 砚池形制选择 | 单选或下拉选择砚池形状（圆形、方形、椭圆形、异形等）            |
| 客户下单页  | 订单提交   | 表单验证、提交成功提示、可选择继续下单                   |
| 管理员订单页 | 订单列表   | 表格展示所有订单，支持分页、排序、搜索                   |
| 管理员订单页 | 工序进度   | 时间轴/步骤条展示生产工序：采石→切坯→整形→雕刻→打磨→上蜡→质检→完工 |

## 3. 核心流程

客户选择砚石材和尺寸 → 选择雕刻样式和砚池形制 → 提交订单 → 管理员查看订单 → 按工序逐步更新生产状态 → 订单完成

```mermaid

    A["客户选择石材"] --> B["输入尺寸"]
    B --> C["选择雕刻样式"]
    C --> D["选择砚池形制"]
    D --> E["提交订单"]
    E --> F["管理员查看订单"]
    F --> G["采石"]
    G --> H["切坯"]
    H --> I["整形"]
    I --> J["雕刻"]
    J --> K["打磨"]
    K --> L["上蜡"]
    L --> M["质检"]
    M --> N["完工"]
```

## 4. 界面设计

### 4.1 设计风格

* 主色调：深棕色 #8B4513，体现传统文房四宝的厚重感

* 辅助色：米黄色 #F5F5DC，营造古典雅致氛围

* 按钮风格：圆角按钮，悬停有阴影效果

* 字体：使用衬线字体增强传统感，标题使用书法风格字体

* 布局：卡片式布局，表单居中对齐

* 图标：使用中国风线性图标

### 4.2 页面设计概览

| 页面名称   | 模块名称 | UI 元素                    |
| ------ | ---- | ------------------------ |
| 客户下单页  | 表单区域 | 卡片容器、标题、下拉选择框、数字输入框、提交按钮 |
| 客户下单页  | 顶部导航 | Logo、页面切换按钮              |
| 管理员订单页 | 订单列表 | 表格、搜索框、分页器               |
| 管理员订单页 | 工序进度 | 步骤条、状态标签、操作按钮            |

### 4.3 响应式设计

* 桌面端优先设计，1200px 以上最佳展示

* 平板端自适应，表单单列布局

* 移动端适配，表格横向滚动
