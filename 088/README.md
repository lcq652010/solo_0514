# gym-management

健身房私教管理系统前端项目，基于 Vue2 + ElementUI 开发。

## 功能模块

- 私教列表页 - 查看所有教练信息，支持筛选和搜索
- 会员约课表单页 - 填写预约信息，表单验证
- 我的约课记录页 - 查看预约记录，管理预约状态
- 课时核销页面 - 支持扫码、预约码、手机号三种方式核销
- 教练排班表页 - 日历式排班管理，周排班一览

## 技术栈

- Vue 2.6.14
- Vue Router 3.5.3
- Element UI 2.15.13
- Sass

## 安装依赖

```
npm install
```

### 开发模式

```
npm run serve
```

### 生产构建

```
npm run build
```
