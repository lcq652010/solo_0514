📁 项目结构

  • package.json - 项目依赖配置
  • vue.config.js - Vue CLI 配置
  • public/index.html - HTML 入口模板
  • src/main.js - Vue 应用入口
  • src/App.vue - 主应用组件（含导航）
  • src/router/index.js - 路由配置
  • src/styles/common.scss - 公共样式
  • src/data/mock.js - 模拟数据
  • src/views/GoodsList.vue - 商品拼团列表
  • src/views/OrderForm.vue - 下单参团表单
  • src/views/LeaderManage.vue - 团长订单管理
  • src/views/Verify.vue - 提货核销页面
  • src/views/Progress.vue - 拼团进度展示

🎯 功能说明

  • 商品拼团列表：展示所有拼团商品，支持按状态和关键词筛选，点击可跳转下单
  • 下单参团表单：完整的表单验证，包含商品选择、数量、收货人、电话、地址等字段
  • 团长订单管理：订单统计卡片，订单列表，搜索筛选，状态更新功能
  • 提货核销页面：输入提货码查询订单，确认提货，核销记录展示
  • 拼团进度展示：进度条可视化，参团成员列表，参团功能
