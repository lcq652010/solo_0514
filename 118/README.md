# 奶茶门店点单系统

基于 Vue2 + ElementUI 开发的奶茶门店点单前端页面

## 功能模块

### 1. 商品分类及规格选择 (Menu.vue)
- 四个商品分类：人气推荐、经典奶茶、鲜果茶、咖啡系列
- 点击商品弹出规格选择对话框
- 可选择：杯型、温度、糖度、加料（珍珠/椰果/芋圆/奶盖）
- 购物车功能：添加、修改数量、删除
- 表单验证确保规格必填

### 2. 顾客点单结算
- 在购物车页面点击"去结算"
- 填写顾客姓名、联系电话、取餐方式
- 支持备注特殊要求
- 订单明细展示
- 表单验证：姓名、手机号必填，手机号格式验证

### 3. 订单列表页 (Orders.vue)
- 按条件搜索订单：订单号、顾客姓名、订单状态
- 订单列表展示：订单号、顾客信息、金额、状态、下单时间
- 查看订单详情
- 取消订单（仅待制作状态）
- 状态标签：待制作(橙色)、制作中(蓝色)、已完成(绿色)、已取消(红色)

### 4. 制作状态更新页 (Production.vue)
- 三栏布局：待制作、制作中、已完成
- 点击"开始制作"将订单移至制作中
- 点击"制作完成"将订单移至已完成
- 显示顾客信息、商品明细、规格、备注
- 实时显示各状态订单数量

### 5. 营业额简易统计卡片 (Statistics.vue)
- 统计卡片：
  - 总订单数
  - 总营业额
  - 今日订单
  - 今日营业额
- 订单状态统计：待制作、制作中、已完成、已取消
- 热销商品 TOP5 排行
- 近期订单明细表格

## 技术栈

- **Vue 2.6.x** - 前端框架
- **Vue Router 3.5.x** - 路由管理
- **Element UI 2.15.x** - UI 组件库
- **SCSS** - 样式预处理
- **localStorage** - 数据持久化

## 安装与运行

### 安装依赖
```bash
npm install
```

### 开发模式运行
```bash
npm run dev
```
访问 http://localhost:8080

### 构建生产版本
```bash
npm run build
```

## 页面路由

| 路径 | 页面 | 说明 |
|------|------|------|
| /menu | 商品点单 | 商品展示、规格选择、购物车、结算 |
| /orders | 订单列表 | 订单查询、详情查看、状态更新 |
| /production | 制作状态 | 三栏状态看板，制作流程管理 |
| /statistics | 营业统计 | 数据统计卡片、热销排行 |

## 数据说明

- 订单数据存储在浏览器 localStorage 中
- 商品数据为模拟数据，包含 12 种奶茶和咖啡
- 支持多种规格组合（杯型、温度、糖度、加料）

## 主要特性

1. **响应式布局** - 适配不同屏幕尺寸
2. **表单验证** - 提交前验证必填项和格式
3. **状态管理** - 订单状态流转清晰
4. **数据持久化** - 刷新页面数据不丢失
5. **用户体验** - 加载状态、成功提示、确认对话框
