# 考试报名系统

基于 Vue2 + ElementUI 开发的完整考试报名系统前端项目。

## 功能模块

1. **考试科目列表** - 查看可报名的考试科目，包含科目信息、考试时间、报名进度等
2. **报名申请表单** - 填写个人信息、选择科目、上传证件照进行报名
3. **资格审核页面** - 管理员对报名申请进行审核通过/驳回操作
4. **准考证生成** - 查询并打印准考证信息
5. **报名记录列表** - 查看个人报名记录及审核状态

## 技术栈

- Vue 2.7.14
- Vue Router 3.6.5
- Element UI 2.15.14
- Webpack 5

## 功能说明

### 1. 考试科目列表
- 展示所有考试科目信息
- 显示考试日期、时间、费用
- 实时显示报名进度条
- 支持按科目名称/代码搜索
- 点击"立即报名"跳转至报名页面

### 2. 报名申请表单
- 选择考试科目（仅显示可报名科目）
- 完整的个人信息填写（姓名、身份证、手机、邮箱等）
- 上传身份证照片和证件照
- 表单必填校验和格式验证
- 提交报名申请

### 3. 资格审核页面
- 按姓名、审核状态筛选报名记录
- 查看报名详情
- 审核通过/驳回操作
- 驳回时可填写原因

### 4. 准考证生成
- 通过身份证号和姓名查询准考证
- 展示完整准考证信息（考场、座位号等）
- 支持打印功能
- 考生须知提示

### 5. 报名记录列表
- 按科目、状态、时间范围筛选记录
- 查看报名详情
- 审核通过后可查看准考证
- 待审核状态可取消报名
