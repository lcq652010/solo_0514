# 布艺香包定制系统

基于 Vue2 + ElementUI 实现的布艺香包定制系统，包含客户下单和管理员订单管理功能。

## 功能特性

### 客户下单页面
- ✅ 客户信息填写（姓名、电话）
- ✅ 面料材质选择（棉麻、丝绸、绸缎、亚麻）
- ✅ 香包造型选择（圆形、方形、心形、三角形、菱形）
- ✅ 尺寸输入（5-20cm）
- ✅ 内料配方多选（薰衣草、艾草、檀香、薄荷、玫瑰、桂花）
- ✅ 挂绳样式选择（中国结、流苏、皮绳、编织绳）
- ✅ 订购数量、备注说明
- ✅ 表单验证
- ✅ 订单提交

### 管理员订单管理
- ✅ 订单统计卡片（待开始、生产中、已完工、总订单）
- ✅ 8道工序管理：裁布 → 缝纫 → 填香 → 收口 → 串珠 → 挂绳装配 → 质检 → 完工
- ✅ 工序进度可视化显示
- ✅ 按工序筛选订单
- ✅ 工序更新操作
- ✅ 订单删除
- ✅ 订单详情展示

## 技术栈

- Vue 2.6.x
- Vue Router 3.5.x
- Element UI 2.15.x

## 安装运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run serve

# 构建生产版本
npm run build
```
