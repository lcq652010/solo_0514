# 快递驿站管理系统

基于 Vue2 + Element UI 开发的快递驿站管理前端系统。

## 功能模块

- **寄件下单** - 快递寄件信息登记，表单校验，运费自动计算
- **取件核销** - 取件码查询，包裹状态展示，取件核销
- **包裹入库** - 快递包裹入库登记，取件码自动生成
- **包裹列表** - 包裹信息查询管理，多条件筛选，分页展示
- **会员管理** - 会员信息增删改查，会员等级管理

## 技术栈

- Vue 2.x
- Vue Router 3.x
- Element UI 2.x
- Webpack 5.x

## 项目结构

```
├── src/
│   ├── views/              # 页面组件
│   │   ├── Layout.vue      # 主布局组件
│   │   ├── SendOrder.vue   # 寄件下单
│   │   ├── PickupVerify.vue # 取件核销
│   │   ├── PackageInbound.vue # 包裹入库
│   │   ├── PackageList.vue # 包裹列表
│   │   └── MemberManage.vue # 会员管理
│   ├── router/
│   │   └── index.js        # 路由配置
│   ├── styles/
│   │   └── common.css      # 公共样式
│   ├── App.vue             # 根组件
│   └── main.js             # 入口文件
├── index.html              # HTML模板
├── webpack.config.js       # Webpack配置
├── package.json            # 项目依赖
└── .babelrc                # Babel配置
```

## 安装和运行

### 安装依赖

```bash
npm install
```

### 开发模式运行

```bash
npm run dev
```

项目将在 `http://localhost:8080` 启动

### 生产环境构建

```bash
npm run build
```

## 页面说明

### 1. 寄件下单
- 寄件人/收件人信息录入
- 物品类型、重量选择
- 快递类型选择（普通/加急/冷链）
- 运费自动计算
- 表单必填项校验

### 2. 取件核销
- 取件码查询包裹
- 手机号验证
- 包裹状态展示
- 取件确认核销
- 今日取件记录展示

### 3. 包裹入库
- 快递单号录入
- 快递公司选择
- 收件人信息登记
- 存放区域、货架编号
- 取件码自动生成
- 包裹状态登记

### 4. 包裹列表
- 多条件查询（单号、姓名、电话、公司、状态）
- 表格展示包裹信息
- 状态标签显示
- 分页功能
- 查看/编辑/删除操作

### 5. 会员管理
- 会员信息增删改查
- 会员等级管理（普通/银卡/金卡/钻石）
- 积分、余额管理
- 会员状态管理
- 多条件筛选查询

## 主要特性

- 响应式侧边栏导航
- 统一的表单校验机制
- 表格数据分页展示
- 弹窗式详情查看
- Element UI 风格统一
